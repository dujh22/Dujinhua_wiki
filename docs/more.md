## 命令

* `mkdocs new [dir-name]` - 创建新项目。
* `mkdocs serve` - 启动重新加载的docs服务器。
* `mkdocs build` - 构建文件网站。
* `mkdocs -h` - 显示帮助文档并退出。

## 项目布局

```
mkdocs.yml    # 配置文件。
docs/
    index.md  # 文件主页。
    ...       # 其他页面，文档，图像。
```
